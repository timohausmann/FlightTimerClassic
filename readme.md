Available commands: 

* `/ftc mute` – Enable/disable information in chat window.
* `/ftc show` – Show the FTC UI for positioning.
* `/ftc hide` – Hide the FTC UI.
* `/ftc debug` – Show/hide debug information about flight map buttons.
* `/ftc help` – Show help.
